Reward system

Below model shows when you recieve rewards after staking.

First Epoch User starts delegating.

1 epoch is

Second Epoch Stake Snapshot. Your delegation is registered.
Third Epoch Delegation is now active. All blocks from here on, you will get rewards from
Fourth Epoch If blocks was produced in last epoch they will be calculated now.
Fifth Epoch Rewards will now be delivered.
