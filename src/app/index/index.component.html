The Cardano Rewards System

The model below displays how and when you receive potential rewards after staking.

Epoch N User starts delegating.

1 epoch equals 5 days.
(5 x 24 hours)
Epoch N+1 Stake Snapshot of the blockchain.

Your delegation is registered.
Epoch N+2 Delegation is now active. You will receive rewards from all minted blocks.
Epoch N+3 If any blocks was minted in the last epoch, they will be calculated.
Epoch N+4 Rewards from minted blocks are distributed.
All epochs after. The network guarantees 4-5% annually in rewards.
